### 2. Security Stack / Pilastra de Segurança

**[EN] Technical Infrastructure:**
- **Encryption:** All data in transit is protected by SSL/TLS 1.3 encryption (A+ Rating).
- **OWASP Top 10 Mitigation:** 
    - **SQL Injection:** Automated parameterized queries and ORM protection.
    - **XSS (Cross-Site Scripting):** Dynamic input sanitization and strict Content Security Policies (CSP).
    - **CSRF:** Token-based validation for all state-changing operations.
- **Data Sanitization:** Multi-layer validation for all public endpoints.

**[PT] Infraestrutura Técnica:**
- **Criptografia:** Todos os dados em trânsito são protegidos por criptografia SSL/TLS 1.3 (Classificação A+).
- **Mitigação OWASP Top 10:** 
    - **SQL Injection:** Consultas parametrizadas automáticas e proteção via ORM.
    - **XSS (Cross-Site Scripting):** Sanitização dinâmica de entradas e Políticas de Segurança de Conteúdo (CSP) rígidas.
    - **CSRF:** Validação baseada em tokens para todas as operações de mudança de estado.
- **Sanitização de Dados:** Validação em múltiplas camadas para todos os endpoints públicos.
